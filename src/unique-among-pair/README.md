# Unique among pair

## Question
- For given array of one unique number and other paired numbers (pair may be placed in random position) your task is to find a unique number.

  ### Example
  ```typescript
    const array = [1, 2, 2, 1, 4, 5, 4] //output 5
  ```

### Function Signature
